## Installation

Install gazebo8 with ([details](http://gazebosim.org/tutorials?tut=install_ubuntu)):

```bash
$ curl -ssL http://get.gazebosim.org | sh
$ gazebo
```

Also, include:
```bash
$ sudo apt-get install ros-kinetic-gazebo8-ros
```
