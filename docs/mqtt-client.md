# MQTT Client Notes

## Python Client Installation

### OSX

Python 2
```bash
$ pip install paho-mqtt
```

Python 3
```bash
$ pip3 install paho-mqtt
```

### Raspbian

```bash
$ sudo apt-get install python3-pip
$ sudo pip3 install paho-mqtt
```
